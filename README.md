# Campus Complaint Management System

A full-stack web application where students can submit complaints and administrators can manage and resolve them efficiently.

## Features
- Submit complaints
- Categorize complaints
- Priority based complaint management
- Admin dashboard
- Real-time complaint status updates

## Tech Stack
Frontend:
- React (Vite)
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL
- Drizzle ORM

## Project Structure

client/   -> React frontend
server/   -> Express backend
shared/   -> shared schemas

## Run Locally

git clone https://github.com/MSUSHANTHREDDY5/Campus-Complaint-Hub.git

cd Campus-Complaint-Hub

npm install

npm run dev
