CORE PRIORITY SYSTEM

Improve complaint prioritization using both voting and an aging mechanism.

priorityScore = votes + agingFactor

agingFactor = hours since complaint was created / 6

This ensures that every 6 hours unresolved complaints gradually increase in priority so older issues are not ignored.

Complaints must automatically sort by priorityScore so the most urgent issues appear first.

Priority Colors:
Green → Low priority
Yellow → Medium priority
Red → High priority
